# Khandaq Messenger Asset Pack

This archive contains a structured production asset pack for Khandaq Messenger, based on the supplied logo and optimized for app stores, desktop platforms, favicons, website, and social media use.

The recommended primary app icon is located at `Brand/Icons/Final/khandaq-icon-1024.png`. For mobile publishing, use the platform-specific folders rather than the original emblem, because the final icon removes text and uses a simplified shield-lock symbol for small-size readability.

For design handoff, import `Brand/Logo/khandaq-icon-editable.svg` into Figma or Illustrator. The SVG is intentionally clean and editable. The original provided PNG/JPG are preserved in `Brand/Logo/Source` for brand continuity.
