Decentralized messenger with no central message server. End-to-end encryption, Khandaq bootstrap nodes, clients for all major platforms.

  • No phone number — Tox ID only
  • Text and files over the Tox P2P network

Installation notes

Android — sideload APK
  1. Download APK and verify SHA256.
  2. Allow installation from your browser (unknown sources).
  3. Open the APK and confirm install.
  4. Upgrading from unsigned debug builds: export Tox profile → uninstall old app → install new APK → import profile (signature change).
Windows — SmartScreen
  1. If warned, click More info → Run anyway.
  2. Installer is not Authenticode-signed during beta.
  3. Verify SHA256 if your antivirus blocks the file.
macOS — Gatekeeper
  1. Unzip and move khandaq.app to Applications.
  2. If blocked: xattr -cr /Applications/khandaq.app
  3. Or: right-click → Open → confirm.
Linux (portable)
  1. tar -xzf khandaq-linux-x86_64-portable.tar.gz
  2. Keep khandaq, khandaq.bin, and lib/ together.
  3. See INSTALL.txt inside the archive.

Quick start

  1. Install a client and create a profile (remember your password).
  2. Copy your Tox ID and share it with a contact.
  3. Add the contact and send a test message.
