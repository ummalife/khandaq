# Figma / Illustrator Handoff

Use `khandaq-icon-editable.svg` as the editable vector source. It can be imported directly into Figma, Adobe Illustrator, Sketch, Affinity Designer, or any vector editor that supports SVG.

The SVG contains the selected Concept B app icon geometry: deep emerald background, fortress-shield outline, lock, keyhole, and simplified security silhouette. The file is intentionally clean and editable rather than a raster trace of the supplied emblem.

Native `.fig` and `.ai` files generally require proprietary editor export. This package therefore includes the cross-platform vector source and matching PDF/PNG exports for design and engineering handoff.
